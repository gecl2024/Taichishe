Adventures With Anxiety!

An interactive story about anxiety - where you play as the anxiety itself

About the Game

Adventures With Anxiety is an interactive story game created by Nicky Case. In this game, you play as "anxiety" itself, experiencing the inner world of someone with anxiety disorder. The game uses humor to show how anxiety works, and might help players reduce their fear of fear itself.

You'll control a hyperactive anxiety wolf whose mission is to protect the human character from various "dangers". Through this interactive story, you'll experience how anxious thinking works and how it affects daily life.

How to Play

  • Click or press spacebar - Advance dialogue/interaction
  • Click on options - Make choices
  • Click anywhere - Pause/resume game

Note: Game contains adult language, mentions of alcohol use, and some sensitive topics

Features

  • A warm and humorous narrative exploring anxiety
  • Unique perspective: play as anxiety itself rather than the person
  • Simple and fun hand-drawn style with a hyperactive wolf character
  • Provides useful information and advice about mental health
  • Multiple branching choices affecting story development
  • Original music by Monplaisir

About the Creator

Nicky Case is an independent game developer and interactive storyteller who focuses on exploring complex psychological and social issues through games. Adventures With Anxiety is created based on the creator's own experience with anxiety disorder.

The game is completely open-source, with all content released under the Creative Commons Zero (CC0) license, allowing anyone to freely use, modify, and share the content.

Game Characters

  • Anxiety Wolf (Player Character) - A hyperactive, overprotective wolf representing the protagonist's anxiety
  • Human Protagonist - An ordinary person facing social, professional, and personal challenges

Source Code

The complete source code for this game can be found on GitHub: Adventures With Anxiety Source Repository

The game uses various open-source technologies, including Howler.js (sound effects), MinPubSub (publish/subscribe), RSVP.js (Promises), and more.
